# Análisis de Materialidad en una Consultora Web

## Introducción
En el desarrollo web actual, no solo se construyen aplicaciones funcionales, sino que también se generan impactos en la sociedad, el medio ambiente y la gestión empresarial. Este proyecto analiza dichos impactos mediante un enfoque de **materialidad ASG (Ambiental, Social y Gobernanza)**.

## Objetivo
Identificar los principales *stakeholders* y los temas materiales clave en una consultora web, representándolos mediante un archivo XML estructurado.

---

## Stakeholders identificados

- **Clientes**: Buscan soluciones eficientes, seguras y sostenibles.
- **Usuarios finales**: Requieren accesibilidad, usabilidad y privacidad.
- **Equipo de desarrollo**: Responsable de implementar buenas prácticas técnicas.

---

## Temas materiales

- **Eficiencia energética del código** (*Ambiental*):
Reducción del consumo energético mediante optimización del software.

- **Accesibilidad universal** (*Social*):
Inclusión de todos los usuarios, siguiendo estándares como *WCAG*.

- **Privacidad de datos** (*Gobernanza*):
Protección de la información personal conforme a normativas como *GDPR*.

---
